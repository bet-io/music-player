## 🌟 项目简介

TuneHub 是一个功能强大的单文件网页音乐播放器，支持从网易云音乐、酷我音乐、QQ音乐等多个平台搜索、播放和下载音乐。V2.0 版本新增了 **9大功能模块**，包括深色模式、音频可视化、智能推荐等！

## ✨ 核心功能

### 🎵 播放功能
- ✅ **多平台搜索**：网易云音乐、酷我音乐、QQ音乐、聚合搜索

## 🎮 使用指南

### 基本操作
1. **搜索音乐**：输入关键词选择平台搜索
2. **播放控制**：点击播放按钮或按空格键
3. **切换歌曲**：使用上一首/下一首按钮
4. **调节音量**：拖动音量滑块（0-100%）

### 高级功能
1. **切换主题**：点击右上角的 🌙/☀️ 图标
3. **创建歌单**：输入名称点击"创建歌单"
4. **添加歌曲**：点击歌曲旁的 📝 按钮
5. **查看统计**：统计区域实时显示播放数据
6. **导出数据**：点击"导出数据"按钮备份

## 🐛 常见问题

### Q: 无法搜索歌曲？
A: 检查网络连接和API可用性，确保能访问 `https://music-dl.sayqz.com`


### Q: 如何添加歌曲到歌单？
A: 1. 创建歌单 → 2. 选择歌单 → 3. 点击歌曲旁的 📝 按钮

### Q: 数据丢失怎么办？
A: 使用"导出数据"功能定期备份，或检查浏览器是否清除了本地存储
