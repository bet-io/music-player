## ✨ 特性

### 🎵 播放功能
- **多平台支持**: 网易云音乐、酷我音乐、QQ音乐、聚合搜索
- **多种音质**: 128k、320k、FLAC、FLAC 24bit
- **实时切换**: 无缝切换音质，保持播放进度
- **歌词同步**: LRC 歌词实时滚动显示
- **键盘快捷键**:
  - `空格键`: 播放/暂停
  - `←/→`: 快退/快进 10 秒

### 🎨 界面设计 (V2.0 新增)
- **深色/浅色主题**: 完整主题切换系统，保护眼睛 🌙/☀️
- **迷你播放器**: 浮动式小窗口，多任务操作不遮挡 ⬇️/⬆️
- **音频可视化**: 实时频谱显示，随音乐节奏变化 🎵
- **现代玻璃拟态**: 使用 backdrop-filter 实现毛玻璃效果
- **响应式布局**: 完美适配桌面和移动设备
- **动态效果**: 播放时专辑封面脉冲动画
- **直观操作**: 一键下载音频、歌词、封面

## 🎯 使用指南

### 1. 搜索音乐
1. 在搜索框输入歌曲名、歌手名或关键词
2. 选择搜索平台（或使用聚合搜索）
3. 点击搜索按钮

### 2. 播放音乐
1. 从搜索结果中选择一首歌曲
2. 歌曲将自动开始播放
3. 使用底部控制栏控制播放

### 3. 切换音质
1. 点击音质选择器
2. 选择想要的音质等级
3. 播放器会自动切换并保持进度

### 4. 下载文件
- **音频**: 点击"下载当前音质"按钮
- **歌词**: 点击"歌词"按钮
- **封面**: 点击"封面"按钮
- **全部**: 点击相应的下载组合按钮

### 5. 调节音量
- 拖动音量滑块调节音量
- 音量设置会自动保存

### V2.0 新功能使用指南

#### 🌙 切换深色模式
- 点击右上角的 🌙/☀️ 图标切换主题
- 设置会自动保存，下次打开时保持

#### ⬇️ 使用迷你播放器
- 点击右下角的 ⬇️ 按钮切换到迷你模式
- 迷你播放器固定在右下角，包含完整控制
- 点击 ⬆️ 返回正常播放器

#### 🎼 选择播放模式
- 在顶部下拉菜单选择播放模式
- 支持顺序、单曲循环、列表循环、随机播放
- 设置会自动保存

#### 📝 管理歌单
1. **创建歌单**: 输入名称，点击"创建歌单"
2. **选择歌单**: 点击创建的歌单进行选择
3. **添加歌曲**: 点击搜索结果旁的 📝 按钮
4. **查看歌曲**: 选中歌单后显示所有歌曲
5. **删除歌曲**: 点击歌曲右侧的 × 按钮

#### 📊 查看播放统计
- 统计区域实时显示播放数据
- 包括总播放次数、今日播放、累计时长等

#### 🧠 使用智能推荐
- 推荐区域基于播放历史生成
- 点击推荐可直接搜索播放

#### 💾 数据管理
- **导出数据**: 点击"导出数据"按钮备份
- **导入数据**: 点击"导入数据"恢复备份
- **清空数据**: 一键清除所有本地数据
