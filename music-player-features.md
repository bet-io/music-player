# TuneHub 音乐播放器增强功能说明

## 🌙 1. 深色模式切换功能

点击右上角的月亮/太阳图标可以切换亮色/深色主题，设置会自动保存到本地存储。

## ⬇️ 2. 迷你播放器模式

- 点击右下角的向下箭头按钮切换到迷你播放器
- 迷你播放器显示当前播放歌曲的基本信息和进度条
- 可以继续控制播放/暂停、上一首/下一首
- 再次点击向上箭头返回完整播放器界面

## 🎵 3. 音频频谱可视化效果

- 使用 Web Audio API 实现实时频谱显示
- 在播放器底部显示动态音频可视化效果
- 使用渐变色彩，随音乐节奏变化

## 🎼 4. 播放模式

在顶部选择播放模式：
- **顺序播放**：按照列表顺序播放
- **单曲循环**：重复播放当前歌曲
- **列表循环**：播放完列表后从头开始
- **随机播放**：随机选择歌曲播放

## 📝 5. 自定义歌单功能

- 创建和管理多个歌单
- 输入歌单名称点击"创建歌单"
- 歌单信息保存在浏览器本地存储中
- 支持删除歌单功能

## 📊 6. 播放统计数据

实时显示播放统计信息：
- **总播放次数**：累计播放歌曲总次数
- **今日播放**：今日播放歌曲数量
- **总时长**：累计播放时长（分钟）
- **最爱歌曲**：标记为喜爱的歌曲数量

## 🧠 7. 智能推荐系统

基于播放历史生成个性化推荐：
- 分析播放频率最高的歌曲
- 根据播放习惯推荐相似歌曲
- 点击推荐歌曲可以直接搜索播放

## 💾 8. 数据管理功能

提供完整的数据导入导出功能：
- **导出数据**：将歌单、统计、设置导出为 JSON 文件
- **导入数据**：从备份文件恢复数据
- **备份数据**：备份歌单为文本文件
- **清空数据**：清除所有本地数据

## 🔔 9. 通知提醒系统

- 操作反馈：切换主题、播放模式等操作会有通知提示
- 播放状态：播放歌曲、下载完成等状态通知
- 自动消失：通知 3 秒后自动消失

## 📁 10. 离线模式

- 所有歌单和统计数据保存在浏览器本地
- 即使关闭浏览器后重新打开，数据依然存在
- 支持离线查看播放历史和歌单

## 🎮 增强快捷键

除了原有的快捷键外，新增：
- `Ctrl+P`: 播放/暂停
- `Ctrl+L`: 下一首
- `Ctrl+K`: 上一首
- `Ctrl+S`: 显示搜索框
- `F`: 全屏（未来版本）

## 🎨 界面优化

- 完整的暗色模式支持
- 平滑的过渡动画
- 响应式设计，支持各种屏幕尺寸
- 玻璃态效果的界面设计

## 🔧 技术特性

- 使用 localStorage 进行数据持久化
- Web Audio API 实现音频可视化
- Canvas 绘制动态频谱
- ES6+ 语法实现现代 JavaScript
- 响应式 CSS Grid/Flexbox 布局

## 📱 兼容性

- 支持现代浏览器（Chrome、Firefox、Safari、Edge）
- 需要 ES6+ 支持
- Web Audio API 支持

## 🚀 使用建议

1. 首次使用建议先搜索几首歌曲，系统会根据您的播放习惯生成推荐
2. 定期导出数据作为备份
3. 尝试不同的播放模式，找到最适合您的方式
4. 使用迷你播放器进行多任务操作

## 📞 问题反馈

如遇到问题，请检查浏览器控制台错误信息，或导出数据文件发送给开发团队进行调试。
